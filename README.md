# dotnet

### DRAWER
MS Paint clone written in .NET WindowsForms featuring:
- saving & loading images
- undo/redo
- resising canvas
- drawing tools:
  - line
  - hand drawing
  - styled text
  - outlines & solid shapes:
    - rectangles
    - elipses
### Calculatron3000
Simple calculator written in .NET WindowsForms featuring:
 - all basic operations
 - x^y power
 - log
 - y root x
 - 1/x
 - sin, cos, tan
 - saving number to memory
